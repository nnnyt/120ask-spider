## Usage
```
$ python3 spider.py
```

## 数据
数据格式：
```json
[
    {
        类型: "检查",
        网址: "http://xxx..."
        名称: "xx",
        简介: "xxx",
        属性: {
        	专科分类: "",
        	检查分类: "",
        	...
    	},
    	正常值: "",
    	临床意义: "",
    	注意事项: "",
    	检查过程: "",
    	不适宜人群: "",
    	不良反应与风险: "",
    	相关疾病: [
    		{名称: "aaa", 网址: ""},
    		{名称: "bbb", 网址: ""},
    		...
		]
    }
]
```
运行后默认存储在当前文件夹data.json文件中，可在调用write_json()函数时通过参数`data_dir`指定目标输出目录
